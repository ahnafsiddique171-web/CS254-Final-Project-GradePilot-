# GradePilot

GradePilot is a student-focused academic planning web app. It helps users estimate their current course grade, calculate the score needed on a final exam, and predict their semester GPA.

## Features

- Add weighted course grade categories
- Calculate estimated course grade
- Calculate required final exam score for a target grade
- Add courses with credit hours and expected letter grades
- Calculate estimated semester GPA
- Save progress using browser local storage
- Reset all saved data
- Responsive modern layout

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser Local Storage
- Cloudflare Pages for deployment

## Technical Challenge

One technical challenge was connecting the course grade estimator and GPA calculator into one dashboard while keeping the data organized. The solution was to store course grade categories and GPA courses in separate JavaScript arrays, then update the dashboard whenever the user adds, deletes, saves, or resets data.

## Future Improvement

A future improvement would be allowing users to create multiple saved semesters or export their academic plan as a PDF.
